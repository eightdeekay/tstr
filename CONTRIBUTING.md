# Contributing to tstr

Thanks for your interest! tstr is a young project and the DSL design is still
evolving, so contributions and ideas are welcome.

## Licensing

By submitting a contribution, you agree that it is licensed under the project's
[MIT License](LICENSE) — the same terms as the rest of tstr (inbound = outbound).

## Before a big change

The language design lives in [`README.md`](README.md) and is still in flux.
For anything beyond a small fix, please **open an issue first** to discuss the
direction — it saves everyone a round of rework.

## Building and testing

```bash
cargo build
cargo test
```

Please make sure `cargo test` passes before opening a pull request.
